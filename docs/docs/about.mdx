#

## JPage: Creating Slide-Based Websites

**JPage** is an [open source](https://github.com/pedro-isacss/jpage) JavaScript library that simplifies the creation of slide-based websites, offering an easy and fast way to create pages with both horizontal and/or vertical slide navigation. Whether you're building a dynamic portfolio, an interactive presentation, or a single-page website, JPage is the ideal tool.

import PreviewJpage from "../src/components/PreviewJpage";

## Key Features

- **Horizontal and Vertical Scrolling**: Native support for navigating between slides vertically or horizontally, with smooth transitions.
- **Keyboard Navigation**: Users can navigate through slides using the keyboard, offering an intuitive user experience.
- **Fully Responsive**: The slide layout automatically adapts to different screen sizes and devices.
- **Custom Transitions**: Configure slide transition time and style to create a personalized visual experience.
